Envato Market Child

WordPress Theme & Plugin management for the Envato Market (Child).

View on GitHub Download .zip

Welcome!

The Envato Market Market plugin can install WordPress themes and plugins purchased from ThemeForest & CodeCanyon by connecting with the Envato Market API using a secure OAuth personal token. Once your themes & plugins are installed WordPress will periodically check for updates, so keeping your items up to date is as simple as a few clicks.

The Envato Child plugin extend Envato Market plugin in installing nested packages along the theme

Installation

Installing the Envato Child plugin.

  1. Download the envato-child.zip and unpack the archive.
  2. Upload envato-child to the /wp-content/plugins/ directory.
  3. Activate the plugin through the 'Plugins' menu in WordPress.
  4. Click the 'Envato Child' menu in WordPress and connect to the API.
